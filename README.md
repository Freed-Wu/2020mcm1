## Code Style ##

## File Type ##

table file type: `csv`

**don't recommend `xls` and `xlsx`**

figure file type: `png`(shotsnap), `jpg`(web download), `pdf`(draw), `ai`(draw),
eps(draw)

**don't recommend `bmp`**

reference file type: `bib`
